# O que é?
```
Uma API simulando uma clinica veterinaria em spring boot
```
### O que tera ?
```
* CRUD de Cliente
* CRUD de paciente (animais)
* CRUD de Veterinarios / funcionarios
* CRUD de fixa tecnica
* Agendamento de consultas
* Autenticação
* Autorização
* Envio de Email
* Relatorios
* Log

```
